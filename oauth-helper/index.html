SpotiRide

Uzyskaj Refresh Token potrzebny do polaczenia Spotify z Garmin Edge 530

Krok 1: Wklej dane ze Spotify Developer

Jesli nie masz jeszcze aplikacji Spotify Developer — instrukcja jest w pliku README.

Wazne! W panelu Spotify Developer w polu Redirect URI musi byc wpisane dokladnie:
https://garmin-spotify-callback.invalid/callback

Krok 2: Zaloguj sie do Spotify

Po kliknieciu przycisku otworzy sie nowa karta z logowaniem do Spotify.

  1. Zaloguj sie swoim kontem Spotify
  2. Kliknij "Zgadzam sie" (zezwol na dostep)
  3. Zobaczysz strone z bledem "nie mozna otworzyc" — to jest normalne!
  4. Skopiuj caly adres z paska przegladarki (Ctrl+A, Ctrl+C w pasku)
